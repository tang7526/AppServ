Welcome to AppServ2015 website.

Since author of Appserv don't update it very long time, I decided to build a new version of appserv called Appserv2015.

Appserv2015 is an open project that is a merging open source software installer package for windows that has easy way to build webserver, databse server.

The installer package includes：

  • Apache 2.4.16
  • PHP 5.6.12
  • MySQL 5.6.26
  • phpMyAdmin 4.4.14

How to install

Double click AppServ2015-1.0.0.exe. Just click next and next and next....until finish.

If you start apacher error, you can try install Visual C++ Redistributable for Visual Studio 2012 Update 4 (x86) and start apache again beause apache and php are compiled by visual stuio 2012.

How to uninstall

Double click uninstall.exe if you want to uninstall appserv2015. It will remove Appserv2015 folder but keep MySQL\data folder and www folder.

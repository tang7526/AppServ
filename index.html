Welcome to AppServ2015 website.

Since author of Appserv don't update it very long time, I decided to build a new version of appserv called Appserv2015.

Appserv2015 is a open project that is a merging open source software installer package for winsows that has easy way to build webserver, databse server.

The installer package includes：

  1. Apache 2.4.16
  2. PHP 5.6.12
  3. MySQL 5.6.26
  4. phpMyAdmin 4.4.14

How to install

Double click AppServ2015-1.0.0.exe. Just click next and next and next....until finish.

How to uninstall

Just double click uninstall.exe if you want to uninstall appserv2015. It will remove Appserv2015 folder and all of fils. Remember to backup files that you want to keep before you execute uninstall.exe.
